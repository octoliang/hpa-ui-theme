# hpa-ui-theme
> element component hpa-ui theme.


## Installation
```shell
npm i hpa-ui-theme -S
```

## Usage

Use Sass import
```css
@import 'hpa-ui-theme';
```

Or Use webpack
```javascript
import 'hpa-ui-theme';
```

Or
```html
<link rel="stylesheet" href="path/to/node_modules/hpa-ui-theme/lib/index.css">
```

##  Import on demand
```javascript
import 'hpa-ui-theme/lib/input.css';
import 'hpa-ui-theme/lib/select.css';

// ...
```
